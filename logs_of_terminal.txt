You can very well customize the authentication system. Please go through the following steps:

Steps:

Create your own user table like and add this in models.py file of your app.
class UserData(models.Model): 
 """ 
 This class saves the user id along with the Address 
 """ 
 
 userid = models.CharField(max_length=10, primary_key=True) 
 username = models.CharField(max_length=50) 
By default Django authenticates the user using the authenticate method of
django.contrib.auth.authenticate  
Since you have your own UserData table you can override the way django authenticates user by creating your own authentication system.

To do this create a backends.py  file in your app and put two methods authenticate and get_user.

from django.contrib.auth.models import User 
from .models import UserData 
 
class MyBackEnd(object): 
 """ 
 This is the custom backend to authenticate the user in the DB. 
 if this authentication fais then django default authentication  will get called 
 """ 
 
 def authenticate(self, userid, password=None): 
 """ 
 :param username: Username of the user 
 :param password: Password of the user 
 :return: 
 """ 
 #check is user is present in User DB. 
  existing_user = User.objects.get(username=userid) 
  if not existing_user: 
   #Checking the user UserData Custom DB. 
   user_data = UserData.objects.get(userid=userid) 
   print("...%s...." % user_data) 
   if userid == user_data.userid: 
    user =  User.objects.create_user(username=userid,password=12345) 
    user.save() 
    return user 
   else: 
    return None 
  else: 
   return existing_user 
 
 def get_user(self, user_id): 
 try: 
  return User.objects.get(username=user_id) 
 except User.DoesNotExist: 
  return None 
Now make the following entry in settings.py file

DEBUG = True 
 
ALLOWED_HOSTS = [] 
 
AUTHENTICATION_BACKENDS = ('loadmap.backends.MyBackEnd', 'django.contrib.auth.backends.ModelBackend') 
 
# Application definition 
Put both the backends, django tries to authenticate from second backend if it fails from the first one.

Do the following entry in your app urls.py file

from django.conf.urls import url 
from django.contrib.auth import views as auth_views 
 
from . import views 
 
app_name='loadmap' 
 
urlpatterns = [url('login/$', auth_views.login, name='login'), 
              .....]

{% extends "admin/base.html" %}

from django.shortcuts import render 
from .models import UserData

 :return:
